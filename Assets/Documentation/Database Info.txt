QuestSystem - 
  Each quest is assigned a unique ID.
  All quests are known to the quest system
  Discovered quests are known by the player.

InventorySystem -
  Each item is assigned a unique ID.
  Each item has a name, description, and use.
  An inventory is a list of these items.
  Each entity knows of its own inventory

CharacterSystem -
  Each character is assigned a unique ID.
  Characters can have different traits: talkable, enemy
